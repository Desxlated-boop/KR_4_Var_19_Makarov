# KR_4_Var_19_Makarov
React-приложение для планирования задач по дням недели.
## Функционал
- Добавление задач в любой день недели (ввод → Enter)
- Удаление задач (кнопка ×)
- Автоматическое сохранение всех задач в localStorage
- Адаптивная таблица с фиксированными столбцами
- Сохранение данных после перезагрузки страницы
